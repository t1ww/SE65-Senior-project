Note on Docker (Production):
In real deployment (for safety):

Run code like this inside Docker containers to isolate file, CPU, memory, etc.

Example Docker run for C++:
    docker run --rm -v /tempdir:/temp gcc:latest g++ /temp/tempfile.cpp -o /temp/exe
